Questions 1. How would you normally debug a node application?

I feel using a VS Code Debugger helps better in debugging as compared to conventional debuggers.

I use the --inspect keyword while starting the app in VS Code to attach a debugger in my code.

2. Which node modules do you commonly use in your toolbelt and why?

For my front end my go to module is the material module, since it makes the styling part very easy.

Where as for my back end development, I use express, and MongoDb as my database.

3. Can you tell us whether you prefer using Directives or Components in AngularJS and why?

Both directives and components have their functions and advantages, if I have to create a view then I would create a component, but if I have to provide a nature or functionality to it, I would use a directive.
